## 🎱 학습 내용 (updating...)
- DefinitelyTyped
  - typescript 지원 커뮤니티
  - react와 react-dom은 타이핑을 지원하지 않기 때문에, 따로 설치해주어야 한다. `@types/react @types/react-dom`

- 타입스크립트 세팅
  - tsconfig.json에 `"esModuleInterop": true,`설정하면, export로 내보내진 React와 ReactDOM을 export default로 내보내진 것처럼 사용할 수 있다.
